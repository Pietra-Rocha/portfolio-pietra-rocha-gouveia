# SM6 - Engenharia de Software e IA com Bubble.io

Este projeto foi desenvolvido como parte da disciplina de Engenharia de Software com Inteligência Artificial, utilizando a plataforma Bubble.io para criação de uma aplicação web funcional de forma prática e intuitiva.

O objetivo principal foi compreender como ferramentas de desenvolvimento visual e recursos de IA podem auxiliar na criação de sistemas modernos, responsivos e centrados na experiência do usuário.

---

## Sobre o Projeto

Durante o desenvolvimento, foram aplicados conceitos de:

- Engenharia de Software
- Modelagem de interfaces
- Desenvolvimento No-Code
- Estruturação de fluxos e lógica de negócio
- Integração de Inteligência Artificial
- Experiência do Usuário (UX/UI)

A aplicação foi construída utilizando recursos visuais do Bubble.io, permitindo criar páginas, componentes interativos, navegação entre telas e automações sem necessidade de programação tradicional.

---

## Tecnologias e Ferramentas Utilizadas

- Bubble.io
- Inteligência Artificial
- UX/UI Design
- Engenharia de Software
- Modelagem de Fluxos
- Prototipação Visual

---

## Objetivos da Atividade

- Explorar o desenvolvimento de aplicações utilizando plataformas No-Code;
- Aplicar conceitos de Engenharia de Software em um projeto real;
- Desenvolver uma interface intuitiva e funcional;
- Entender como a IA pode auxiliar no processo de desenvolvimento;
- Criar fluxos de navegação e interação entre páginas.

---

## Funcionalidades Desenvolvidas

- Interface visual interativa;
- Navegação entre páginas;
- Componentes responsivos;
- Estrutura organizada de telas;
- Fluxos automatizados;
- Experiência otimizada para o usuário.
