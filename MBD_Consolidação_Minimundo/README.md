# Sistema de Avaliação e Cadastro dos Melhores Comércios do Brasil

## Descrição
Este projeto consiste na definição e consolidação do "Minimundo", um sistema voltado para o cadastro, avaliação e ranqueamento de estabelecimentos comerciais em todo o território nacional. O objetivo é promover transparência e confiabilidade no mercado local.

## Objetivos
* Gerenciar informações cadastrais de comércios.
* Implementar sistema de comentários e avaliações por usuários.
* Gerar relatórios e rankings de performance por categoria e localização.
