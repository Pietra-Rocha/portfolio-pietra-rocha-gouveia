# 🧩 Artefato 05 – Normalização do Banco de Dados

## 🧠 Objetivo
Este artefato valida o modelo de dados aplicando as formas normais (1FN, 2FN e 3FN).

## 📊 Formas Normais Aplicadas

### ✔ Primeira Forma Normal (1FN)
- Dados atômicos
- Sem listas ou repetições
- Chave primária definida

### ✔ Segunda Forma Normal (2FN)
- Dependência total da chave primária
- Eliminação de dependências parciais

### ✔ Terceira Forma Normal (3FN)
- Sem dependências transitivas
- Redução de redundância
- Maior consistência dos dados

## 🗂 Tabelas Validadas
- Comércio
- Categoria
- Usuário

## 🔍 Resultados
Todas as tabelas:
✔ Atendem 1FN  
✔ Atendem 2FN  
✔ Atendem 3FN  

## 🔄 Revisão do DER
Após a normalização:
- Estrutura otimizada
- Relações ajustadas
- Redução de inconsistências

## 🎯 Conclusão
O banco de dados está preparado para implementação, garantindo integridade e eficiência.

## 📄 Documento
Este artefato comprova a qualidade do modelo lógico através das formas normais.
