# Artefato 02 – Dados vs Informação

## Objetivo
Este artefato apresenta conceitos fundamentais sobre dados e informação dentro do contexto do sistema de comércios.

## Conceitos Abordados

### Dado vs Informação
- **Dado:** valor bruto (ex: 1499.90)
- **Informação:** dado com contexto (ex: preço de um produto em promoção)

## Exemplos
- Valor de produto → informação de venda  
- Data → informação logística  
- Código → identificação de produto  

## Tipos de Dados

### Dados Estruturados
- Número do pedido  
- Categoria  
- Endereço  

-> Possuem formato definido e fácil armazenamento

### Dados Não Estruturados
- Avaliações de clientes  

-> Texto livre, exige análise

## Importância
Esses conceitos são essenciais para:
- Modelagem de banco de dados  
- Organização de informações  
- Tomada de decisão  

## Documento
Este artefato auxilia na compreensão da transformação de dados em informação útil no sistema.
