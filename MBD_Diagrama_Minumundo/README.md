# Modelagem de Dados - Sistema de Avaliação de Comércios

## Descrição
Documentação do modelo de banco de dados (DER) para o sistema de avaliação de comércios. Este repositório contém a estrutura lógica das entidades necessárias para suportar as operações do Minimundo.

## Entidades Principais
* **Comércio**: Informações principais do estabelecimento.
* **Assinatura**: Gestão de planos dos comerciantes.
* **Relatórios/Status**: Dados operacionais e administrativos.
* **Produtos/Serviços**: Ofertas vinculadas aos comércios.
* **Localização**: Dados geográficos vinculados ao comércio.
