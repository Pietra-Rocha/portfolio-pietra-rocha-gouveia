# Estudo de Caso: Eventos de Gala S.A.

## Descrição
Análise de requisitos e modelagem de dados para o sistema da empresa "Eventos de Gala S.A.". 
O foco foi a estruturação das entidades e relacionamentos para o gerenciamento de eventos.

## Entidades Analisadas
* Cliente, Evento, Local, Funcionário, Fornecedor, Serviço, Tarefa e Fatura.

## Relacionamentos Chave
* Cliente solicita Evento (1:N).
* Evento acontece em Local (1:N).
* Tarefa atribuída a Funcionário (N:1).
* Evento gera Fatura (1:1).
