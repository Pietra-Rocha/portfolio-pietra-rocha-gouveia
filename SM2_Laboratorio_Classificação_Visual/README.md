# Laboratório de Classificação Visual - Teachable Machine

## Descrição
Projeto acadêmico utilizando o Teachable Machine para criar um modelo de visão computacional capaz de distinguir pessoas com e sem boné através da webcam.

## Integrantes
* Pietra Rocha Gouveia
* Thiago Messias Fernandes

## Considerações sobre Ética e IA
* **Análise de Viés:** Discussão sobre como a seleção restrita de dados de treinamento (apenas com/sem boné) gera viés e exclusão.
* **Ação Mitigadora:** Aplicação do método *Human-in-the-loop* para revisão de dados e aumento da diversidade das amostras (diferentes acessórios, luz e ângulos).
